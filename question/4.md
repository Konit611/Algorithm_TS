問題 1: パスワードバリデーション (Validation)
ユーザーが入力したパスワードが、システムのセキュリティポリシーに適合するか判定する関数を作成してください。

要件
1. 結果を返すための型 PasswordResult を定義すること。
- isValid: boolean
- errors: string[]（エラーがない場合は空配列）

2. 関数 validatePassword(password: string): PasswordResult を作成すること。

3. バリデーションルール：
- 8文字以上であること。
- 大文字を1文字以上含むこと。
- 数字を1文字以上含むこと。

4. 期待値
```typescript
/**
 * [入力値, 期待されるisValid, 期待されるエラーメッセージ数]
 */
const passwordTestData = [
  ["Pass1234", true, 0],       // 正常系
  ["pass1234", false, 1],      // 大文字なし
  ["Password", false, 1],      // 数字なし
  ["Ab1!",     false, 1],      // 8文字未満
  ["",         false, 3],      // 全エラー（空文字）
  ["12345678", false, 1],      // 大文字なし
  ["ABCDEFGH", false, 1],      // 数字なし
] as const;
```
